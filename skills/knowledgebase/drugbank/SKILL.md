# DrugBank - Drug Knowledge Base Skill

## About DrugBank

DrugBank combines detailed drug data with comprehensive drug target information. It includes:
- Drug (small molecule and biotech) information
- Target proteins
- Mechanisms of action
- Drug interactions

## Access

The open version (DrugBank 5.0) is available via XML download. This skill can work with a local downloaded copy.

## Usage

Get comprehensive drug information from DrugBank including:
- Drug description
- Mechanism of action
- Targets
- Drug interactions
- Brand names
