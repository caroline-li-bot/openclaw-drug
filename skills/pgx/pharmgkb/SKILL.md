# PharmGKB - Pharmacogenomics Knowledgebase Skill

## About PharmGKB

PharmGKB is the Pharmacogenomics Knowledgebase, containing:
- Genotype-drug response relationships
- Clinical dosing guidelines
- Pathway information

Website: https://www.pharmgkb.org/

## API Access

PharmGKB provides API access with a free API key available at:
https://www.pharmgkb.org/api

## Usage

Query pharmacogenomic information:
- What dosing guidance is there for clopidogrel with CYP2C19?
- What are the PGx guidelines for warfarin?

### Example Queries:
- "What pharmacogenomic guidance exists for clopidogrel?"
- "CYP2C19 genotype and clopidogrel response"
