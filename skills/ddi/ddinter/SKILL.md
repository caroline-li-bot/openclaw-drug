# DDInter - Drug-Drug Interaction Skill

## About DDInter

DDInter is an online drug-drug interaction database containing:
- ~2k drugs
- ~36k DDIs
- Severity annotation

Website: https://ddinter.scbdd.com/

## API / Access

DDInter provides a web interface, and the data can be downloaded from their website.
This skill queries the DDInter data for drug-drug interactions.

## Usage

Check for interactions between two or more drugs.

### Example Queries:
- "Are there interactions between warfarin and ibuprofen?"
- "What are the interactions between aspirin and clopidogrel?"
